## üöÄ Quick Commands

### Start the System
```bash
# Terminal 1
cd backend && npm run dev

# Terminal 2
cd institution-compass && npm run dev
```

### Test Single Analysis
```bash
curl -X POST http://localhost:5000/api/sentiment/analyze \
  -H "Content-Type: application/json" \
  -d '{"text": "Test complaint"}'
```

### Test Batch Analysis
```bash
curl -X POST http://localhost:5000/api/sentiment/analyze/batch \
  -H "Content-Type: application/json" \
  -d '{"texts": [{"text": "First"}, {"text": "Second"}]}'
```

### Submit Test Complaint
- Go to `http://localhost:5173`
- Submit a complaint
- Log in to admin dashboard
- View complaint details
- See AI Analysis card
